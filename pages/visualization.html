Visualization

EXOSUITES recommends using PlotJuggler for most real-time robotics data visualization. PlotJuggler is widely used within the robotics and ROS communities and provides powerful tools for plotting time-series data.

EXOSUITES includes built-in client functionality that allows controllers to stream data directly to PlotJuggler over a local network connection.

Installing PlotJuggler

PlotJuggler can be installed on the robot itself or on a separate computer connected to the same network.

Linux and Windows are officially supported. macOS users can build PlotJuggler from source.

Note

We recommend installing PlotJuggler on a separate development computer rather than on the robot.

Setting Up PlotJuggler

EXOSUITES uses UDP communication to stream data. Inside PlotJuggler:

  1. Open PlotJuggler.
  2. Select the Streaming tab.
  3. Choose UDP from the available protocols.
  4. Enter your local IP address.
  5. Choose a port number greater than 1024.
  6. Select JSON as the Message Protocol.
  7. Add a timestamp field.
  8. Click OK.

Finding Your IP Address

Use the following commands depending on your operating system:

Windows

```

ipconfig
```

Linux

```

ifconfig
```

or

```

ip a
```

Important Configuration Settings

Setting Value
Protocol UDP
Message Format JSON
Port Any value above 1024
Timestamp Recommended

Network Requirements

Important

PlotJuggler can only communicate with devices located on the same local network.

Internet-based connections are not supported.

Some managed enterprise or domain networks may block UDP communication.

Using PlotJuggler in EXOSUITES

Import the visualization tools: 

```

from exosuites.toolbox import PlotJugglerUDPClient
from exosuites.toolbox import TimedLoop
import time
import math
```

Create a PlotJuggler client: 

```

pj_streamer = PlotJugglerUDPClient(
addr='192.168.0.15',
port=5556
)

loop = TimedLoop(30)
```

Streaming Data Example

```

while loop.continue_loop():

```
data_to_send = {

    'example_data': {

        'sine': math.sin(time.time()),
        'cosine': math.cos(time.time())

    },

    'timestamp': time.time()

}

pj_streamer.send(data_to_send)

This example streams sine and cosine wave data continuously to PlotJuggler.

Why Use a Timestamp?

Including a timestamp field ensures smooth plotting and accurate time synchronization.

Without timestamps, PlotJuggler will use packet arrival times, which may introduce jitter or timing errors.

Alternative Visualization Tools

Although PlotJuggler is recommended, the following tools may also be useful:

Verification Checklist

  • PlotJuggler installed successfully.
  • UDP communication enabled.
  • Correct IP address configured.
  • Port configured properly.
  • JSON protocol selected.
  • Timestamp included.
  • Data visible in PlotJuggler.
