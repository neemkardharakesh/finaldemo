Single Board Computers

This library is designed to be used on Linux computers, typically with GPIO capability. The primary supported platforms are Raspberry Pi and NVIDIA Jetson Orin Nano. These devices provide the processing power required for robotic control, sensing, machine learning, and communication with EXOSUITES hardware.

⚠ Attention

Please read through the entire section for your selected device before beginning setup.

Raspberry Pi

Required Components

  • Raspberry Pi
  • MicroSD Card
  • Computer with Raspberry Pi Imager
  • CAN HAT
  • HDMI Cable
  • Keyboard + Mouse
  • Monitor
  • Power Supply

Basic Setup

Install Raspberry Pi OS using Raspberry Pi Imager. The Desktop version is recommended for beginners because it provides a graphical user interface and development tools.

Complete the operating system setup, connect to Wi-Fi, and update all packages before continuing.

Set Up CAN Bus Capability

If you plan to use CAN-enabled actuators, install and configure a CAN HAT such as PiCAN2, PiCAN3, or Waveshare CAN HAT. 

```

dtparam=spi=on
dtoverlay=mcp2515-can0,oscillator=12000000,interrupt=25
```

Save the configuration file and reboot the Raspberry Pi. Verify that the CAN interface is detected successfully.

Jetson Orin Nano

Required Components

  • Jetson Orin Nano
  • MicroSD Card
  • CAN Transceiver
  • DisplayPort Cable
  • Keyboard + Mouse
  • Monitor
  • Power Supply

Jetson Basic Setup

  1. Flash JetPack OS onto a MicroSD card.
  2. Insert the card into the Jetson and complete first boot configuration.
  3. Connect to the internet and install updates.
  4. Install ROS 2, Python dependencies, and EXOSUITES software.

Enable CAN Communication

Connect a CAN transceiver to the Jetson CAN interface. Verify that the CAN device is detected before connecting actuators.

⚠ Caution

Be careful when accessing the carrier board. Avoid damaging the fan cable and GPIO connectors.

Verification Checklist

  • Operating System boots successfully.
  • Internet connection is working.
  • CAN interface is detected.
  • Python environment is configured.
  • EXOSUITES software is installed.
  • Device is ready for actuator integration.
