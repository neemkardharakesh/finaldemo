IMUs

This section handles all setup steps needed for the supported IMU types: MicroStrain, OpenIMU, and MPU-9250. Although each one requires a different configuration method, EXOSUITES makes interacting with them through Python essentially identical. You can also use any combination of these sensors together in the same setup without issues.

MicroStrain

The low-level driver library for MicroStrain IMUs is maintained by HBK MicroStrain and is routinely updated. However, the installation is not automatically included with the EXOSUITES installation.

EXOSUITES can still be used without it, though the MicroStrain-specific IMU functions and tools will not be available.

Using the EXOSUITES MSCL Installation Helper

EXOSUITES includes a helper script that automatically installs the MSCL dependency.

Run: 

ep-install-mscl

To install into a virtual environment: 

ep-install-mscl -E

After installation, all MicroStrain IMU functionality becomes available.

Manual MSCL Installation

Download the appropriate MSCL package and install it using: 

sudo dpkg -i MSCL__Python_v.deb

If you use a virtual environment, copy the files into the environment: 

cp /usr/share/python3./dist-packages/*mscl*

Verify communication using: 

ep-stream-microstrain-imu --imu-serial-id [SERIAL_ID]

Replace SERIAL_ID with the final six digits of the IMU serial number.

OpenIMU

OpenIMUs require per-unit setup but provide excellent documentation and support resources.

These devices communicate using CAN and should follow the CAN configuration instructions for your single-board computer.

Flashing New OpenIMUs

  1. Connect OpenIMU to a computer using UART.
  2. Provide 12–24V power.
  3. Use an RS232-to-USB adapter.
  4. Download the latest firmware.
  5. Open Google Chrome.
  6. Launch ACEINNA Development Studio.
  7. Select "Get Started".
  8. Download the ACEINNA Web Server.
  9. Run the server and install firmware.

MPU-9250

The MPU-9250 IMU series is easy to use but requires proper I2C bus configuration before operation.

Checking I2C on NVIDIA Jetson Orin Nano

Scan the I2C bus using: 

i2cdetect -y -r 7

For the second bus: 

i2cdetect -y -r 1

If required: 

sudo i2cdetect -y -r 7

Configuring I2C on Raspberry Pi

  1. Open Raspberry Pi Configuration.
  2. Or run:
sudo raspi-config
  1. Select Interface Options.
  2. Select I2C.
  3. Enable I2C.
  4. Save settings.
  5. Restart the Raspberry Pi.

Verification Checklist

  • MSCL installed successfully.
  • MicroStrain IMU detected.
  • OpenIMU firmware updated.
  • CAN communication functioning.
  • I2C interface enabled.
  • MPU-9250 detected correctly.
  • All IMUs communicating with EXOSUITES.
